# Uninstallation
This step below will remove **akaBot Studio** from your device.

**Step 1** - In order to **uninstall** akaBot, open **Control Panel** by typing in Search bar

![image-20220506111003-1.png](/static/img/image-20220506111003-1.png)


**Step 2** - Choose **View By: "Large icons"** and select **"Programs and Features"**

![image-20220506111112-2.png](/static/img/image-20220506111112-2.png)

**Step 3** - Next, **double click** on **akaBot Platform** to **uninstall**

![image-20220506111126-3.png](/static/img/image-20220506111126-3.png)
