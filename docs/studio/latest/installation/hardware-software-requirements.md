# Hardware & Software Requirements

## Hardware Requirements

In order to use akaBot Studio & Agent, your computer needs to fulfill below hardware requirements:

|  | Pre-requisites | Recommended |
|---|---|---|
| **CPU** | 2 × 1.8 GHz 64-bit | 4 × 2.4 GHz 64-bit |
| **RAM** | 4 GB available system memory | 8 GB of available system memory |

## Software Requirements

In order to use akaBot Studio & Agent, your computer needs to fulfill below software requirements:

| | Supported Versions |
|---|---|
| **Operating System** | Windows 10<br/>Windows 10N<br/>Windows 11<br/>Windows Server 2016<br/>Windows Server 2019<br/>Windows Server 2022<br/>Windows Server 2025 |
| **.NET Framework** | Minimum: 4.5.2 |
