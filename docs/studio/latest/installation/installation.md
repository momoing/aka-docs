# Installation

## Before you Start

1. akaBot Studio & Agent are included in the **same** installer. This guideline is used for installing both components in both **Standard** (or **Standalone**) & **Enterprise** (or **Network**) **Edition**.

2. Kindly make sure your computer fulfilling the **hardware and software requirements** before installing.

3. Kindly make sure that you had **an akaBot licensing email** including the Installer download link. If not yet, request a **2-month trial license** for temporary use.

#### 2. Detailed Steps

**Step 1** - Download the installer from the link provided in the Licensing Email

**Step 2** - Extract the zipped file and run the setup as Administrator.


![1689060128056-950.png](/static/img/1689060128056-950.png)

If your computer has already had akaBot platform, go to step 3, otherwise, go to step 4.

**Step 3** - Select Remove akaBot Platform option then hit Finish button to remove the previous akaBot edition.

![image-20230711142249-1.png](/static/img/image-20230711142249-1.png)

Go back to step 2 to re-start the installing.

**Step 4** - Hit Next button to start the installing.

![image-20230711142249-2.png](/static/img/image-20230711142249-2.png)

**Step 5** - Read the license agreement carefully. If you accept the terms, select “I agree” then “Next”, otherwise, hit “Cancel”.

![image-20230711142249-3.png](/static/img/image-20230711142249-3.png)

**Step 6** - Select installation folder.

1 – By default, akaBot will install in folder “C:\Program Files\FPT Software\akaBot Platform”. You can change the clicking “Browse” button then select a folder.

2 – If you prefer that anyone (any profile) on your computer can use akaBot, select “Everyone”, otherwise, select “Just me”.

3 – Hit “Next”

![image-20230711142249-4.png](/static/img/image-20230711142249-4.png)


**Step 7** – Confirm the installation setting by hitting “Next” button.

![image-20230711142249-5.png](/static/img/image-20230711142249-5.png)

**Step 8** - Wait 1-2 minutes for installation to progress.

After successfully installing, hit “Close” button to exit.

![image-20230711142249-6.png](/static/img/image-20230711142249-6.png)

Now you should have the shortcuts for **akaBot Studio & Agent** presenting on your computer.

![image-20230711142249-7.png](/static/img/image-20230711142249-7.png)

**Next Steps**

- You need to **activate the Studio and Agent** to unlock the features.
- You may need to **install the akaBot web extension** for akaBot to automate on web browsers.
