# Akabot Studio

Akabot Studio is the visual workflow designer where you build, test, and publish automation workflows. It provides a drag-and-drop canvas, a rich activity library, and built-in debugging tools so developers and citizen automators can create robust bots without writing boilerplate code. 

## Key features

- **Visual workflow canvas** — Drag, drop, and connect activities to model any business process.
- **Rich activity library** — Hundreds of built-in activities for UI automation, file handling, data manipulation, web services, and more.
- **Integrated debugger** — Step through executions, inspect variable values, and set breakpoints directly inside the designer.
- **Version control** — Commit and push workflows to Git repositories for team collaboration.
- **Publish to Center** — Package and publish workflows to Akabot Center with a single click.
