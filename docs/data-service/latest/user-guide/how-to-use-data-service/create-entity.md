# Create Entity 

All objects created in Data Service are referred to as **Entities**.

**Step 1**: To begin, click on the "Add Entity" button

![image-20230630130806-1.png](/static/img/f5881a_image-20230630130806-1.png)

**Step 2**: Provide the required information.

![image-20230630130815-2.png](/static/img/5c5a7f_image-20230630130815-2.png)

**Step 3**: Once you have entered the information, click on the "Save" button.

![image-20230630130845-3.png](/static/img/b4284e_image-20230630130845-3.png)

**Note**: Check Create Another if you want to create add another Entity without closing pop-up
