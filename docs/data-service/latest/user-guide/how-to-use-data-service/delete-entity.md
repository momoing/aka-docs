# Delete Entity

**Step 1**: To delete an Entity, click on the three dots located next to the Entity that you wish to remove then select "Delete" option

![image-20230630131159-1.png](/static/img/708254_image-20230630131159-1.png)

**Step 2**: Confirm the deletion by clicking on the "Delete" button.

![image-20230630131206-2.png](/static/img/24d5e4_image-20230630131206-2.png)
