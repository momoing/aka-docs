# Update Field 

**Step 1:** Start by selecting the Entity to which you want to edit field. On Field tab, click on the three dots icon located next to the Field that you want to update

![image-20230630131504-1.png](/static/img/bb3902_image-20230630131504-1.png)

**Step 2:** Make the necessary changes

![image-20230630131515-2.png](/static/img/09e670_image-20230630131515-2.png)

**Step 3:** Click [Save] button to save modifications
