# Edit Entity 

**Step 1**: Start by clicking on the three dots located next to the Entity that you want to edit.

![image-20230630131024-1.png](/static/img/083974_image-20230630131024-1.png)

**Step 2**: Edit the Entity Name according to your requirements.

![image-20230630131031-2.png](/static/img/83055d_image-20230630131031-2.png)

**Step 3**: Once you have made the necessary changes, click on the "Save" button to save the modifications.
