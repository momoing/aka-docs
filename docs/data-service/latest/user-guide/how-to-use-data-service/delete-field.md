# Delete Field

**Step 1:** Start by selecting the Entity to which you want to delete field. On Field tab, click on the three dots icon located next to the Field that you want to delete then choose Delete option

![image-20230630131628-1.png](/static/img/4c8163_image-20230630131628-1.png)

**Step 2**: Confirm the deletion by clicking on the "Delete" button.

![image-20230630131633-2.png](/static/img/fbdd56_image-20230630131633-2.png)
