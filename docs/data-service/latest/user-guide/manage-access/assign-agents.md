## **Assign Agents**

You can assign agents and set their permissions by following the below steps:

Step 1: In **Manage Access**, select**Agents** tab​

![image-20230630133429-2.png](/static/img/578e5c_image-20230630133429-2.png)

Step 2: Click **Assign Agents**

![image-20230630133527-3.png](/static/img/f25045_image-20230630133527-3.png)​

Step 3: In the opened popup, select desired agents and their permissions. ​

![image-20230630133612-4.png](/static/img/0f3afe_image-20230630133612-4.png)

Step 4: Click **Save**.

The newly added agents are displayed in the **Agents** list of the **Manage Access** tab.

## **Edit Agents**

You can edit the custom permissions of an agent by selecting **Action icon** button, then clicking **Edit**.

![image-20230630133736-5.png](/static/img/397f1c_image-20230630133736-5.png)

​

## **Remove Agents**

You can remove an agent from Data Service by selecting **Action icon** button, then clicking **Remove**.

![image-20230630134235-7.png](/static/img/af67d0_image-20230630134235-7.png)
