## **Assign Tokens**

You can assign token by following below steps:

Step 1: In **Manage Access**, select**Tokens** tab​

![image-20230630134349-1.png](/static/img/083db7_image-20230630134349-1.png)

Step 2: Select **Assign Token**

![image-20230630134438-2.png](/static/img/2c817a_image-20230630134438-2.png)

Step 3: In the opened popup, select desired tokens. ​

![image-20230630134510-3.png](/static/img/ae2fa5_image-20230630134510-3.png)

Step 4: Click **Save**.

The newly added tokens are displayed in the **Tokens** list of the **Manage Access** tab.

## **Edit Token**

You can edit token permissions by selecting **Action icon** button, then clicking **Edit.**

![image-20230630134716-4.png](/static/img/9802ef_image-20230630134716-4.png)​

## **Remove Token**

You can remove a token from Data Service by selecting **Action icon** button, then clicking **Remove.**

![image-20230630134832-5.png](/static/img/a3c7b1_image-20230630134832-5.png)
