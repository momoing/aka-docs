## **Assign user**

You can assign users and set their permissions by following below steps:

Step 1: In **Manage Access**, select**Users** tab​

![image-20230630134940-1.png](/static/img/9fa9b1_image-20230630134940-1.png)

Step 2: Select **Assign User** ​

![image-20230630135024-2.png](/static/img/9a20cf_image-20230630135024-2.png)

Step 3: In the opened popup, select desired users and their permissions. ​

![image-20230630135056-3.png](/static/img/12cd60_image-20230630135056-3.png)

Step 4: Click **Save**.

The newly added users are displayed in the **Users** list of the **Manage Access** tab.

## **Edit user**

You can edit custom permissions by selecting **Action icon** button, then clicking **Edit**

![image-20230630135237-4.png](/static/img/337315_image-20230630135237-4.png)​

## **Remove user**

You can remove a user from Data Service by selecting **Action icon** button, then clicking **Remove**

​![image-20230630135327-5.png](/static/img/1345da_image-20230630135327-5.png)
