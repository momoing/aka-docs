# System Requirements

## akaBot Center (Server)

| Component | Minimum | Recommended |
|---|---|---|
| OS | Windows Server 2016 / Ubuntu 20.04 | Windows Server 2022 / Ubuntu 22.04 |
| CPU | 4 cores | 8 cores |
| RAM | 8 GB | 16 GB |
| Disk | 50 GB SSD | 200 GB SSD |
| Database | SQL Server 2016+ / PostgreSQL 12+ | SQL Server 2022 / PostgreSQL 15+ |

## akaBot Agent (Workstation)

| Component | Minimum |
|---|---|
| OS | Windows 10 / Windows Server 2016+ |
| CPU | 2 cores |
| RAM | 4 GB |
| Disk | 10 GB |
| .NET | .NET Framework 4.7.2+ |
