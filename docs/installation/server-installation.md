# Server Installation

## Download

Download the akaBot Center installer from the [support portal](https://support.akabot.com).

## Install on Windows

1. Run `akabotcenter-setup.exe` as Administrator
2. Accept the license agreement
3. Choose installation path
4. Configure the database connection string
5. Set the admin password
6. Click **Install** — the service starts automatically

## Verify

Open a browser and navigate to `http://localhost:8080`. The akaBot Center login screen should appear.
