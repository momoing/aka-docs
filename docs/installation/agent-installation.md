# Agent Installation

The akaBot Agent runs on target machines where bots execute.

## Install

1. Download `akabot-agent-setup.exe` from akaBot Center → **Agents → Download Agent**
2. Run as Administrator
3. Enter the **akaBot Center URL** and **Machine Key** during setup
4. Complete the installation — the agent connects automatically

## Verify Connection

In akaBot Center, go to **Agents** — the new agent appears with status **Available**.
