# License Activation

## Online Activation

1. Log in to akaBot Center as Administrator
2. Go to **Settings → License**
3. Enter your **License Key**
4. Click **Activate** — the license is validated online

## Offline Activation

If the server has no internet access:
1. Generate a **Machine Fingerprint** from the License page
2. Send the fingerprint to support@akabot.com
3. Apply the returned activation file via **Import License**
