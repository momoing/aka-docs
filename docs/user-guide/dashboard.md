# Dashboard

The akaBot Center dashboard provides a real-time overview of your automation environment.

## Widgets

- **Active Bots** — number of currently running bot processes
- **Job Queue** — pending and completed jobs in the last 24 hours
- **Error Rate** — failed job percentage over time
- **Resource Usage** — agent CPU and memory utilization
