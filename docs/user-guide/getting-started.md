# Getting Started

This guide walks you through creating and running your first bot.

## Prerequisites

- akaBot Studio installed on your workstation
- akaBot Agent installed and connected to akaBot Center
- A valid akaBot license

## Step 1: Create a New Process

1. Open **akaBot Studio**
2. Click **New Process** → enter a name and description
3. The Workflow Designer opens with an empty canvas

## Step 2: Add Activities

Drag activities from the **Activities Panel** onto the canvas. See [Activities](./workflow-designer/activities) for a full reference.

## Step 3: Run the Bot

Click **Run** (F5) to execute the process locally in Studio, or publish it to akaBot Center and schedule it via [Bot Management](./bot-management/create-bot).
