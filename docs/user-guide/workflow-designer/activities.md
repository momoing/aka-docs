# Activities

Activities are the building blocks of akaBot workflows. Each activity performs a single action.

## Categories

- **UI Automation** — click, type, read screen elements
- **File & Folder** — read, write, move, delete files
- **Data Manipulation** — string, list, dictionary operations
- **Web** — HTTP requests, browser automation
- **Database** — query, insert, update records
- **Control Flow** — if/else, loops, try/catch, delay
