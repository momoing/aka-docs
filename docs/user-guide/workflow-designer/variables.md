# Variables

Variables store data that flows between activities in a workflow.

## Variable Types

| Type | Description |
|---|---|
| `String` | Text values |
| `Int32` | Integer numbers |
| `Boolean` | True/false |
| `DataTable` | Tabular data |
| `List<T>` | Ordered collections |
| `Dictionary<K,V>` | Key-value pairs |
