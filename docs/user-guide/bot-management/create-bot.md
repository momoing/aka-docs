# Create a Bot

After developing your process in akaBot Studio, publish it to akaBot Center.

## Publish from Studio

1. Open your process in akaBot Studio
2. Go to **File → Publish**
3. Select the target akaBot Center environment
4. Click **Publish** — the process appears in Center under **Processes**

## Register as a Job

In akaBot Center, go to **Processes** → select your process → click **Create Job** to run it on demand.
