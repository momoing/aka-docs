# Schedule a Bot

Use **Triggers** in akaBot Center to run bots on a schedule.

## Create a Trigger

1. Go to **Center → Triggers → New Trigger**
2. Select the **Process** and target **Agent**
3. Set the **Cron expression** or use the visual scheduler
4. Click **Save** — the trigger activates immediately
