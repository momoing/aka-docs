# Publish a Custom Activity

## Via akaBot Center

1. Go to **Center → Activity Packages → Upload**
2. Upload the `.nupkg` file
3. In akaBot Studio, go to **Manage Packages** → the package appears under **Local Feed**
4. Install it — activities appear in the Activities panel

## Via Local NuGet Feed

Configure a local NuGet source in Studio's package manager pointing to your package directory.
