# SDK Overview

The akaBot SDK enables developers to extend the platform by building **custom activities**, integrating with external systems, and automating workflows programmatically.

## Available SDKs

| SDK | Language | Use Case |
|---|---|---|
| [.NET SDK](./dotnet-sdk/getting-started) | C# / VB.NET | Custom activities, Windows automation |
| [Python SDK](./python-sdk/getting-started) | Python 3.8+ | Data processing, API integrations |

## Custom Activities

Build reusable activities that appear in akaBot Studio's activity panel. See [Create a Custom Activity](./custom-activities/create-activity).
