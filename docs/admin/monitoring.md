# Monitoring

## Job Logs

All job executions are logged. View logs at **Center → Jobs → [Job ID] → Logs**.

## System Health

The **Dashboard** shows real-time agent status, job throughput, and error rates.

## Alerts

Configure email or webhook alerts for job failures at **Settings → Alerts**.

## Metrics Export

akaBot Center exposes a Prometheus-compatible `/metrics` endpoint for integration with Grafana.
