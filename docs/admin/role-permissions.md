# Roles & Permissions

## Built-in Roles

| Role | Description |
|---|---|
| **Administrator** | Full access to all features |
| **Developer** | Can publish and manage processes |
| **Operator** | Can run jobs and view logs |
| **Viewer** | Read-only access |

## Custom Roles

Create custom roles with fine-grained permissions at **Admin → Roles → New Role**.
