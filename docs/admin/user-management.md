# User Management

## Create a User

1. Go to **Admin → Users → New User**
2. Enter name, email, and assign a role
3. The user receives an email invitation

## Deactivate a User

Go to the user's profile → click **Deactivate**. The user loses access immediately but data is retained.

## Active Directory / LDAP

akaBot Center supports AD/LDAP authentication. Configure the LDAP connection in **Settings → Authentication**.
