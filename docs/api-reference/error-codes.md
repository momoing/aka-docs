# Error Codes

| Code | HTTP Status | Description |
|---|---|---|
| `AUTH_001` | 401 | Invalid or expired token |
| `AUTH_002` | 403 | Insufficient permissions |
| `PROCESS_001` | 404 | Process not found |
| `PROCESS_002` | 409 | Process already running |
| `AGENT_001` | 404 | Agent not found |
| `AGENT_002` | 503 | Agent unavailable |
| `JOB_001` | 404 | Job not found |
| `QUEUE_001` | 404 | Queue not found |
| `RATE_001` | 429 | Rate limit exceeded |
